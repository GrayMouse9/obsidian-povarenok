# 🍳 Поварёнок — книга рецептов с ИИ для Obsidian

Превращает Obsidian в красивую книгу рецептов с ИИ-помощником. Галерея карточек с обложками, категориями, тегами и оценками; генерация и правка рецептов, распознавание с фото и по ссылке, подсчёт калорий. Работает на компьютере и телефоне.

Модели подключаются через **Groq** (бесплатно, без карты) или любой OpenAI-совместимый API.

## Возможности

- **Книга рецептов** — галерея карточек: обложка, категория, теги, время, звёзды-оценка, отметка «приготовлено». Фильтры по категории, оценке и готовке.
- **Чат-помощник** — пишет и правит рецепты; бережно меняет только то, о чём просят.
- **Рецепт из фото** — распознаёт рецепт с одной или нескольких фотографий (vision-модель).
- **Рецепт по ссылке** — вытягивает рецепт с веб-страницы.
- **По шаблону** — новые рецепты в едином формате по твоему образцу.
- **Причесать рецепт** — приводит к аккуратному формату и правит орфографию.
- **Калории** — ИИ считает калорийность по ингредиентам.
- **Голосовой ввод**, вставка фото из галереи, работа на мобильном.

## Установка через BRAT (рекомендуется)

1. В Obsidian установи плагин **BRAT** (Obtainable via Community Plugins).
2. BRAT → **Add Beta Plugin** → вставь ссылку на этот репозиторий → Add.
3. Включи **Поварёнок** в разделе Community plugins.

## Установка вручную

Скопируй `main.js`, `manifest.json`, `styles.css` в папку
`<хранилище>/.obsidian/plugins/ai-note-editor/` и включи плагин в настройках.

## Настройка (обязательно)

Плагину нужен свой API-ключ — **у каждого свой**:

1. Зайди на **console.groq.com**, войди через Google, раздел **API Keys** → **Create API Key** (карта не нужна).
2. Скопируй ключ (`gsk_...`).
3. Obsidian → Настройки → **Поварёнок** → вставь ключ в поле **API Key**.

Модели по умолчанию (Groq): GPT-OSS 120B → Llama 3.3 70B → GPT-OSS 20B (перебираются, если одна занята). Распознавание фото — Llama 4 Maverick. Всё меняется в настройках. Groq иногда переименовывает модели — если какая-то устареет, укажи актуальную из console.groq.com/docs/models.

## Совет: новые заметки сразу как рецепты

Любая новая **пустая** заметка в папке рецептов автоматически превращается в заготовку рецепта. Чтобы встроенная кнопка «+» Obsidian всегда создавала заметки именно там:

**Настройки → Файлы и ссылки → «Папка для новых заметок» → «В указанной папке» → выбери папку рецептов** (по умолчанию `Рецепты`).

Тогда и кнопка «＋ Создать рецепт» в книге, и системный «+» Obsidian будут давать рецепт.

## Приватность

Текст рецептов и фото отправляются только выбранному тобой провайдеру (по умолчанию Groq). Ключ хранится локально в твоём хранилище. Никому не передавай свой `data.json` — в нём лежит ключ.
